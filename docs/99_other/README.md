## ブランチルール

* ブランチ名の付け方: {Prefix}/{システム名}/{実装概要}
  * e.g.) feature/user_api/create_users

* Prefix一覧

| Branch Name |   Description    |
| :---------- | :--------------- |
| master      | メイン (Default) |
| develop     | 開発用           |
| release     | 本番用           |
| feature     | 機能の追加・編集 |
| bugfix      | 機能の修正       |
| refactor    | リファクタ       |
| test        | テストの作成     |
| docs        | 仕様書の更新     |
