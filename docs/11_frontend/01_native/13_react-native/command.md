## 初期設定

0. (nodeをインストールしてない場合)
   > https://nodejs.org/ja/download/

1. expo-cliのインストール
   > $ yarn global add expo-cli

2. ディレクトリの移動
   > $ cd `<presto-payのroot>`/native/react

3. パッケージのインストール
   > $ yarn

## アプリの起動方法

1. アプリの起動
   > $ yarn start

2. iOS, Android端末からアプリを開く
   * スマホがPCと同じWi-Fiに接続されていること確認
   * Expoアプリの起動
   * アプリにアクセス

## その他コマンド

* コードのフォーマットチェック
  > $ yarn lint

* コードのフォーマット修正
  > $ yarn lint --fix
